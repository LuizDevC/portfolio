### 🚔 Gerador de BOPM
Sistema web desenvolvido para agilizar a geração de boletins de ocorrência em servidores de Roleplay policial, permitindo preencher informações da ocorrência e gerar relatórios padronizados.

**Tecnologias**
- React
- TypeScript
- Node.js
- Supabase

---

### 📋 Sistema de Relatório de Serviço
Aplicação web criada para automatizar o registro de patrulhamento em servidores RP.  
Antes do sistema, o preenchimento era feito manualmente. Agora o formulário registra as informações e envia automaticamente os dados para o Discord via Webhook.

**Destaques**
- Automação de relatórios
- Integração com Discord
- Interface rápida e organizada
